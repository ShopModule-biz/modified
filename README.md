# Welcome to the heidelpay payment extension for xtcommerce 3 / modified commerce



## Currently supported payment methods:

* credit card
* debit card
* prepayment
* Sofort
* PayPal
* direct debit
* iDeal
* Giropay
* EPS
* invoice

### SYSTEM REQUIREMENTS

The extension requires PHP 5.6;
